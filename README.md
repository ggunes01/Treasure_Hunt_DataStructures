# Treasure Hunt Data Structures

A Java Swing treasure hunt game developed as a 2nd year, 2nd semester Data Structures course project.

The project demonstrates the use of basic data structures such as linked lists, doubly linked lists, and binary search trees in a simple game environment.

## About the Project

Treasure Hunt Data Structures is a desktop game built with Java Swing. The player enters a name, rolls a dice, moves across the game board, collects or loses points depending on the chest type, and saves the final score.

The game includes two levels and a scoreboard screen where player scores can be searched and displayed.

## Features

- Java Swing based graphical user interface
- Player name input
- Dice-based movement system
- Two playable levels
- Score calculation system
- Score saving with `score.txt`
- Scoreboard screen
- Binary Search Tree based score searching
- Linked List and Doubly Linked List usage for game board logic

## Data Structures Used

### Linked List

Used in the first level to represent the game path and store chest values.

### Doubly Linked List

Used in the second level to allow forward and backward movement on the game path.

### Binary Search Tree

Used in the scoreboard system to store and search player scores efficiently.
